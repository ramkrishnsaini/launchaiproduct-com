### Concept B — `images/logo-concept-b-wordmark.svg`
**Description:** Pure wordmark — **Launch** + **AI** (accent) + **Product**. No icon; works at small sizes (LinkedIn profile, email).  
**Color usage:** White `#f4f4f5` / zinc-100 for Launch + Product; **AI** in `#00df82`.

### Concept C — `images/logo-concept-c-monogram.svg`
**Description:** Tight **L** shape + forward arrow / path — suggests “launch” and forward motion without cliché rocket emoji.  
**Color usage:** Dark panel, `#00df82` primary stroke, `#03624c` secondary detail.  
**Typography:** Same as B when used with full name beside it.

**Usage:** Website + social avatars: use **A or C** at 320×320px minimum; use **B** where a square icon is awkward and horizontal text fits.

## D. Social media design system (Canva-friendly)

| Element | Rule |
|--------|------|
| **Typography** | Headlines: bold geometric sans, tight tracking. Body: 16–18px, line-height 1.5, zinc-300/400 on dark. No script fonts. |
| **Post layout** | Safe zone: 64px margin on 1080px canvas. One idea per post. Headline → 2–4 short lines → divider → CTA line. |
| **Background** | Solid `#030f0f` or subtle gradient `#030f0f` → `#0a1a18`. Optional grid at 5% opacity of `#00df82`. |
| **Card style** | Rounded 16–24px, border `1px solid #03624c` at 40–60% opacity, inner padding 32px. |
| **Accent usage** | `#00df82` for keywords, CTAs, bullets — never full paragraphs in green. |
| **CTA style** | Primary: “WhatsApp” or “Book a call” in `#00df82` on transparent or subtle `#03624c` pill. |
| **Icons** | Line icons, 1.5px stroke, white or `#00df82`. No cartoon, no stock “robot” clichés. |

**Canva:** Create brand kit with hex colors above; upload `post-square-1080.svg` as template background; duplicate for each post.

- **Objection “too expensive”:** “We scope to practical investment — and we avoid API-heavy traps where possible. Happy to talk fit on a call.”
- **“Is this a course?”:** “No — consulting + build support for your product, not content.”

### Lead conversion
1. **Bio link** → one landing section (WhatsApp + booking).  
2. **LinkedIn:** Comment with value → DM permission → move to WhatsApp for depth.
